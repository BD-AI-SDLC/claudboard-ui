Variant B · dedicated screen behind a sidebar item

Setup is its own destination

A dedicated "Setup" item in the sidebar replaces Project Health as the gateway for new repos. It hosts a 3-step stepper (Analyse → Generate → Workflow) with expanded current-step detail, file outputs, and clear "imported from terminal" callouts. Once setup is done, the sidebar item collapses (you can still re-open it for Refresh / Tech debt). Project Health becomes a clean health/metrics view.

Strengths
Project Health stays clean for the "everything's fine" case
More room per step — better for first-run education
Stepper makes the DAG impossible to miss
Trade-offs
A whole extra screen + a sidebar slot
User has to context-switch between Setup ↔ Health
