Strengths
Live stream stays visible alongside
Answer out of order, see total scope
Batch submit feels fast for short interviews
Trade-offs
Less conversational; no per-Q dialogue
Awkward for follow-up questions later

Variant C · interview is just another event in the stream

Inline stream cards

Question cards live in the live event stream alongside tool calls and agent events. Answered questions collapse to a one-liner; the current one expands. Reinforces the mental model: an interview is just a particular kind of agent event — which makes follow-up questions in any later phase feel natural.

Strengths
Unifies all agent communication in one place
Works for any phase, any agent
Tight scroll behavior; no panel mode-switch
Trade-offs
Can get lost in noisy streams
No bird's-eye view of total questions
